# Objects

Physical things in the Writer's room. Each object can be examined by the player and provokes one or more ideas (Observations). Objects are sticky — once examined, they're done.

Objects carry world-building. They tell the player about the Writer, the world, and the war without exposition.

## Status Key
- **DRAFT** — placeholder, needs discussion
- **APPROVED** — agreed on, ready to write observation text
- **DONE** — observation text written

## Notes

- Descriptions should be concrete and sensory, not analytical. The object is what it IS, not what it MEANS. Meaning is the player's job.
- Each object needs: what it looks like, what condition it's in, and one detail that implies a history.
- An object can provoke different observations depending on what the player is already thinking about (future feature), but for now each object has a fixed set of observations it provokes.
- The room should feel lived-in, cluttered, and specific to this person's life.
